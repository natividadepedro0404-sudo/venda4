# HYPEX Admin Panel User Guide

## Overview

The HYPEX admin panel provides a centralized interface for managing your online store. This guide covers essential operations and features available to administrators.

## Accessing the Admin Panel

1. Visit `/pages/admin.html` in your browser
2. Log in with your admin credentials
3. The admin panel will display if you have admin privileges

## Features

### Orders Management

- **View Orders**: All orders are listed with their status and details
- **Update Status**: Change order status using the dropdown:
  - "Pedido feito" (Order placed)
  - "Em separação" (Processing)
  - "Enviado" (Shipped)
  - "Entregue" (Delivered)
- **Save Changes**: Click "Salvar" to update order status

### Products Management

- **View Products**: Products are displayed in a grid with images and details
- **Add Product**:
  1. Click "+ Novo Produto"
  2. Fill in required fields:
     - Nome (Name)
     - Descrição (Description)
     - Preço (Price)
     - Estoque (Stock)
  3. Upload product images (up to 5)
  4. Click "Salvar Produto"

- **Edit Product**:
  1. Click "Editar" on any product
  2. Modify fields as needed
  3. Click "Salvar Produto"

- **Delete Product**:
  1. Click "Excluir" on any product
  2. Confirm deletion when prompted

### Coupons Management

- **View Coupons**: All active and expired coupons are listed
- **Add Coupon**:
  1. Click "+ Novo Cupom"
  2. Fill in required fields:
     - Código (Coupon Code)
     - Tipo (Type): Percentage or Fixed amount
     - Valor (Value)
     - Data de Expiração (Expiry Date)
  3. Click "Salvar Cupom"

- **Edit Coupon**:
  1. Click "Editar" on any coupon
  2. Modify fields
  3. Click "Salvar Cupom"

- **Delete Coupon**:
  1. Click "Excluir" on any coupon
  2. Confirm deletion

## Best Practices

1. **Product Images**:
   - Use high-quality images
   - Keep image sizes under 2MB
   - Recommended dimensions: 800x1000px

2. **Coupon Codes**:
   - Use uppercase letters and numbers
   - Keep codes simple and memorable
   - Set reasonable expiration dates

3. **Order Management**:
   - Update order status promptly
   - Double-check status changes
   - Monitor pending orders regularly

## Troubleshooting

If you encounter issues:

1. Check your internet connection
2. Verify you're logged in as admin
3. Try refreshing the page
4. Clear browser cache if needed
5. Contact technical support if problems persist

## Security Notes

- Never share your admin credentials
- Log out after admin sessions
- Regularly review order history
- Monitor failed login attempts
- Report suspicious activity
